# Test Report Thumbnails

This folder contains thumbnail images for the tiff-reducer test report.

## Contents

### Image Types

Each test image has up to 3 thumbnail variants:

| Suffix | Description |
|--------|-------------|
| `_orig.png` | Original input image (256px width) |
| `_comp.png` | Compressed output image (256px width) |
| `_diff.png` | Difference image (black = identical, colored = differences) |

### Total Thumbnails

- **554+ thumbnail images**
- **~185 test images** with full comparison sets
- Generated using GDAL (`gdal_translate -outsize 256 0`)

## Usage

These thumbnails are referenced in:

To view thumbnails on GitHub:

## Generation

Thumbnails were generated with:

```bash
# Original image thumbnail
gdal_translate -outsize 256 0 input.tif thumbnail_orig.png

# Compressed image thumbnail
gdal_translate -outsize 256 0 compressed.tif thumbnail_comp.png

# Difference image (generated by report script)
# Shows pixel-by-pixel differences
```

## File Naming Convention

```
{original_filename}_{type}.png

Where:
  {original_filename} = Original TIFF filename
  {type} = orig | comp | diff
```

Example: `bali.tif_orig.png`, `bali.tif_comp.png`, `bali.tif_diff.png`
